# Kiosk ISS Traker and live launches
An easy to use space-themed live web page.
Could be used for decoration on a kiosk (setting up an old smatphone or tablet with kiosk wiew on chrome), as screensaver or whatever you want.

## Tech stack
The two files are essentially html files. The project was made as an experiment with the assistance of AI.

## How to run
Start the main folder with a basic web server (e.g., using Python, VS Code Live Server, or Node.js) and open `index.html` or `kiosk-iss-launches.html` in your browser.
